If You Can’t Get Flutter to Work with Your Beta Version of Xcode

You need to explicitly set the beta version of Xcode as your default one via the command line: sudo xcode-select -s /Applications/Xcode-beta.app/Contents/Developer/ Afterwards, getting your app to run via flutter run should work as it used to before.
